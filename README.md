# 时光寄语 - 公开资源

本仓库包含时光寄语 App 的公开资源文件。

## 文件目录

### 隐私政策
- [docs/privacy_policy.html](docs/privacy_policy.html) - HTML 版隐私政策
- [docs/privacy_policy.md](docs/privacy_policy.md) - Markdown 版隐私政策

### 分享图片
- [assets/share.png](assets/share.png) - 应用分享二维码/宣传图片

## 访问链接

### 隐私政策
GitHub Pages: https://maifuture.github.io/countdown-widget-static/docs/privacy_policy.html

### 分享图片
直接访问：https://raw.githubusercontent.com/maifuture/countdown-widget-static/main/assets/share.png

## 使用方法

### 在 App 中引用分享图片
```dart
// 从 GitHub 下载分享图片
final imageUrl = 'https://raw.githubusercontent.com/maifuture/countdown-widget-static/main/assets/share.png';
```

## 部署说明

本仓库使用 GitHub Pages 托管静态资源：
1. 启用 GitHub Pages：Settings → Pages → Source: main branch → Save
2. 访问 URL: https://maifuture.github.io/countdown-widget-static/
